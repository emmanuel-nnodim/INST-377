HTML TABLE

An HTML table is defined with the < table> tag. Each table row is defined with the < tr> tag. A table header is defined with the < th> tag. A table data/cell is defined with the < td> tag To make a cell span more than one column, use the colspan attribute. If you want the borders to collapse into one border, add the CSS (border-colapse) property. To define a special style for a special table, add an id attribute to the table.
