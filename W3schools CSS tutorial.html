This is a tutorial on how to use CSS

CSS is a language that describes the style of an HTML document and how HTML elements are displayed.

Css Stands for Cascading style sheets.External style sheets are stored in CSS files

CSS Structure

CSS Syntax.
The selector points to the HTML element you want to style. The declaration block contains one or more declarations seperated by semicolons. Each declaration includes a CSS property name and a value, seperated by a semicolon. A CSS declaration always end with a semicolon, and the declaration blocks are surrounded by curly braces.

The id selector uses the id attribute of an HTML element to select a specific element. The id should be unique within a page since it's used to select one unique element. An id name cannot start with a numberThis paragraph utilizes an id selector.

The class selector selects elements with a specific class attribute. To select class elements write a (.) period followed by the class name. You can also specify when an HTMl elements should be affected by writing (p.center or h1.center)

HTMl elements can also refer to more than one class.

If you have elements with the same style definitions it will be better to group the selectors. EX: h1,h2,p{.....}

A CSS comment starts with /* and ends with */

There are three ways of inserting a CSS style sheet

  1. External style sheet
  2. Internal style sheet
  3. Inline style

With external style sheet, you can change the look of an entire website by changing just one files Each page must include a reference to the external style sheet file insde the < link> element which goes inside the < head> section Link syntax goes as follows: < link rel="stylesheet" type="text/css" href="mystyle.css"> The style sheet must be saved with a .css extension
An internal style sheet may be used if one single page has a unique style and written inside the head section with the <'style'> attribute. An inline style may be used to apply a unique style for a single element like < p> or < h1>

When it comes to the Cascading order an inline style has the highest priority, and will override external and internal styles.

The background-image property specifies an image to use as the background of an element. By default the image is repeated so it covers the entire element. When using a background image, use an image that does not disturb the text. Check the < head> tag for more info.

CSS border properties allow you to specify the style, width, and color of an element's border. The border-style property specifies what kind of border to display.

The following values are allowed for border styling with CSS

  • dotted-Defines a dotted border

  • dashed-Defines a dashed border

  • solid-Defines a solid border

  • double-Defines a double border

  • groove-Defies a 3d ridged border. The effect depends on the border color value

    • The CSS border-width property specifies the width of the four borders

  • ridge-Defines a 3d ridged border. The effect depends on the border color value

  • inset-Defines a 3d inset border. The effect depends on the border color value

  • outset-Defines a 3d outset border. The effect depends on the border-color value.

  • none-Defines no border

  • hidden-Defines a hidden border

  • mixed borders can have from one to 4 values

The CSS border-radius property is used to add rounded borders to an element. You can also specify all the individual border properties for just one side EX:(border-left/right, border-bottom/top)

CSS margin are used to create space around elements. The properties for specifying margins for each element includes:(margin-top/bottom, margin-right/left). Negative values are allowed.

The CSS padding property is used to generate space around an element's content.
